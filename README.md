# ML4D_-Heart_Disease
This project was completed as part of a university course and was carried out in a group.

The goal of the project was to analyse a heart disease dataset and explore which factors may contribute to the presence of heart disease. We used data processing, exploratory data analysis and machine learning techniques to identify patterns in patient data and build a predictive model.

The analysis included data cleaning, feature exploration, visualisation and model training. Variables such as age, cholesterol level, blood pressure, chest pain type and maximum heart rate were analysed to better understand their relationship with heart disease.

In this project, I was responsible for data cleaning and feature exploration. This included preparing the dataset for analysis, checking for missing or inconsistent values, and exploring the most relevant variables for predicting heart disease.
